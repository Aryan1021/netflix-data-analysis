# 📊 Netflix Data Analysis

## 📌 Project Overview
This project analyzes Netflix's dataset to uncover insights about content distribution, growth trends, genres, and production countries.

---

## 🎯 Objectives
- Analyze Movies vs TV Shows distribution
- Identify top content-producing countries
- Understand content growth over time
- Explore genres and ratings
- Extract meaningful insights from data

---

## 🛠️ Tools & Technologies
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

---

```text
netflix-data-analysis

```

---

## 📊 Key Insights

- Netflix has more Movies than TV Shows
- Content production increased rapidly after 2015
- USA and India are top content producers
- Drama and International genres dominate
- Most content is targeted towards mature audiences

---

## 📂 Dataset

Dataset is taken from Kaggle:

https://www.kaggle.com/datasets/shivamb/netflix-shows

Download and place it inside the `data/` folder.

---

## 🚀 How to Run

1. Clone the repository  
2. Install dependencies  
3. Run the notebook  

```bash
pip install -r requirements.txt
jupyter notebook
```
